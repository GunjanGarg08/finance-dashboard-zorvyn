# Finance Dashboard UI

A modern, responsive, and interactive finance dashboard built to help users track, analyze, and manage their financial activity.

This project was developed as part of a frontend assignment, with a strong focus on UI/UX design, component architecture, and state management.

## Features

### Dashboard Overview
- Summary cards for **Total Balance, Income, and Expenses**
- Clean and visually appealing UI with trend indicators
- Responsive layout for all screen sizes

---

### Data Visualization
- **Balance Trend Chart** (time-based)
- **Expenses by Category (Donut Chart)** with center total display
- Interactive tooltips and legends

---

### Transactions Management
- View all transactions with:
  - Date
  - Category
  - Amount
  - Type (Income / Expense)
- **Add, Edit, Delete transactions** (Admin only)
- Real-time UI updates

---

### Filtering & Search
- Search transactions by category
- Filter by:
  - Type (Income / Expense)
  - Category (Dynamic)
- Sorting by date and amount

---

### Role-Based UI
- **Viewer**: Can only view data  
- **Admin**: Can add, edit, and delete transactions  
- Role switching implemented via UI toggle

---

### Insights Section
- Top spending category
- Monthly expense comparison
- Total transactions count
- Average transaction value
- Smart insights based on data

---

### Dark Mode
- Fully functional dark/light theme toggle
- Consistent theming across all components

---

### Data Persistence
- Data stored using **localStorage**
- Ensures data remains across sessions

---

### Responsive Design
- Optimized for:
  - Desktop
  - Tablet
  - Mobile devices

---

## Tech Stack

- **Frontend:** React.js (JavaScript)
- **Styling:** Tailwind CSS
- **Charts:** Recharts
- **State Management:** React Context API
- **Deployment:** Vercel

```bash
# Clone the repository
git clone https://github.com/your-username/your-repo

# Navigate to project folder
cd finance-dashboard

# Install dependencies
npm install

# Run development server
npm run dev
```
